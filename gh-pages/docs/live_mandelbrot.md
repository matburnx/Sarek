# Live Mandelbrot

> **This page describes a superseded approach.** The Thebe/Binder-based live demo has been replaced by the native browser Learn course.

## Run the Mandelbrot Kernel in Your Browser

The **[Interactive Learn Course](/Sarek/learn/)** includes a hands-on Mandelbrot lesson where you modify the kernel and see the rendered image update — running on your own GPU via WebGPU, straight from the browser.

```
[Learn Course →](/Sarek/learn/)
```

The course covers: vector addition, image filters, and the Mandelbrot set — each with an editable Sarek kernel, automatic correctness checking, and GPU execution via WebGPU.
