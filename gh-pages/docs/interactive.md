# Interactive Sarek

> **This page describes a superseded approach.** The Jupyter/Binder integration described here has been replaced by a native browser experience.

## Try Sarek in the Browser — No Installation Required

Sarek now runs directly in your browser via WebGPU:

- **[Playground](/Sarek/playground.html)** — Paste any Sarek kernel and instantly see the generated WGSL/CUDA/OpenCL/GLSL/MSL source code. No account, no install, runs on your GPU.
- **[Interactive Learn Course](/Sarek/learn/)** — Step-by-step GPU programming lessons where you edit a Sarek kernel and run it on your own GPU straight from the page. Covers vector addition, Mandelbrot generation, image filters, and more.

Both options use the same kernel DSL and type system as the OCaml library — code you write in the browser is identical to code you would run natively.
